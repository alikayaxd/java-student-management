# Student Management System

A simple Java-based student management system developed using OOP principles.

## Features
- Add student
- List students
- Update student
- Delete student

## Technologies
- Java
- Eclipse IDE

## Project Structure
- model: Student entity
- dao: Data access layer
- service: Business logic
- ui: User interface
- main: Application entry point

## How to Run
1. Clone the repository
2. Open the project in Eclipse
3. Run the Main class
